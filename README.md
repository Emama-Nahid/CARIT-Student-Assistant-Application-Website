# CARIT Student Assistant Application Website

This project is a PHP-based website for the CARIT Center for Applied Research in Information Technology. It includes multiple informational pages, an eligibility-check feature, and a student assistant application form.

The website was developed as part of an IT 5443 Web Development project. It uses reusable PHP include files for the header, navigation menu, and footer to keep the site organized and consistent across pages. Eligible students can access the application form, submit their information, and view a confirmation page generated through PHP form processing.

## Features

- Multi-page CARIT website
- Reusable PHP header, navigation, and footer
- Student eligibility check
- Student assistant application form
- PHP form processing and confirmation table
- Custom CSS styling for a clean and consistent layout

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
